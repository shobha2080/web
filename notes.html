UI/UX NOTES

What is UI & UX?

User Interface (UI) is the visual aspect of a digital product, including layout, typography, and interactive elements.


User Experience (UX) focuses on making interactions smooth, intuitive, and satisfying for users.


Core Principles of UI Design

  • Principles of UI Design

  • Principles of UX Design

  • Process of UI Design

Structure

Organize content for intuitive navigation and user flow.

Simplicity

Keep designs clean to enhance usability and focus.

Visibility

Ensure important elements are easily noticed and accessible.

Feedback

Provide clear responses to user actions for better interaction.

Tolerance

Design to accommodate user errors and offer recovery.

Reuse Principle

Maintain consistency by reusing UI components to enhance efficiency.

Scale

Use different sizes to create emphasis, guide focus, and improve readability.

Visual Hierarchy

Organize elements in a way that directs users' attention to key content first.

Balance

Maintain a symmetrical or asymmetrical layout for a visually stable interface.

Contrast

Improve readability and focus by using strong color, size, and shape differences.

Gestalt Principles

Use grouping, alignment, and proximity to create a cohesive and intuitive design.
